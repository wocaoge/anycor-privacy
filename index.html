磨素（Anycor）隐私政策

更新日期：[2026-04-03]  |  生效日期：[2026-04-03]

磨素（Anycor）是一款纯本地的个人信息管理工具，我们深知你存储的信息对你至关重要。本隐私政策将帮助你了解我们如何处理你的数据。

核心承诺：你存储在磨素中的密码、证件号码、银行卡信息等所有内容数据，仅保存在你的设备本地，经 AES-256 加密保护，不会上传至任何服务器。

一、我们如何处理你的数据

磨素是一款无账号、无云端的本地工具。你在 APP 中录入的所有信息（包括条目名称、账号、密码、证件号码、备注等）均使用 AES-256-GCM 加密算法加密后存储在你的 iPhone 本地，加密密钥由 iOS 系统安全芯片（Keychain）保管。

这意味着：

  • 你的内容数据不会离开你的设备
  • 我们没有服务器存储你的任何信息
  • 即使设备丢失，没有你的设备密码，他人也无法访问你的数据
  • 卸载磨素后，所有数据将彻底清除，无法恢复

我们基于你的明示同意处理上述信息。你可以随时撤回同意，撤回不影响撤回前已进行的处理活动的合法性。

你在磨素中存储的密码、证件号码、银行卡号等属于敏感个人信息。我们处理这些信息的目的是帮助你集中管理和安全存储个人重要信息。这些信息全程不离开你的设备，仅由你本人通过 APP 访问。如果这些信息泄露，可能导致你的财产安全或人身安全受到影响，因此我们采用 AES-256-GCM 加密算法进行保护。

二、第三方 SDK 信息

为了了解 APP 的使用情况并改进产品体验，磨素集成了以下第三方 SDK。该 SDK 不会获取你存储在 APP 中的任何内容数据。

项目 说明
SDK 名称 友盟+统计分析 SDK（U-APP）
开发者 友盟同欣（北京）科技有限公司
收集的信息 设备型号、操作系统版本、APP 版本、网络状态、语言和时区、会话时长等
用途 统计分析用户使用行为，优化产品功能
隐私政策 友盟+隐私权政策

友盟 SDK 在无网络环境下不会影响 APP 的任何功能，采集的数据将在网络恢复后上报。友盟 SDK 采集的数据存储于中国境内服务器，不会传输至境外。我们已与友盟签署数据处理协议，要求其按照约定的目的和方式处理相关数据。友盟 SDK 采集的数据按其隐私政策的规定保留和处理。

三、数据安全

我们采取以下措施保护你的数据安全：

  • 所有敏感信息使用 AES-256-GCM 加密算法加密存储
  • 加密密钥由 iOS Keychain 保管，仅在设备解锁时可访问
  • 加密密钥不会同步到其他设备，不会备份到 iCloud
  • APP 支持 4 位数字密码和 Face ID / Touch ID 保护
  • APP 切换到后台时自动遮盖屏幕内容

四、你的权利

你对自己的数据拥有完全的控制权：

  • 查看和编辑：你可以随时在 APP 内查看和修改你存储的所有信息
  • 删除数据：你可以在 APP 内删除任意条目，也可以通过卸载 APP 彻底清除所有数据
  • 撤回同意：你可以在 APP「设置」页面中撤回对统计数据采集的同意，撤回后将停止友盟 SDK 的数据采集，APP 核心功能不受影响
  • 投诉举报：如果你认为我们的个人信息处理行为侵害了你的合法权益，你有权向履行个人信息保护职责的部门投诉举报

五、未成年人保护

磨素不会主动收集任何用户（包括未成年人）的个人内容数据。如果你是未成年人，请在监护人指导下使用本 APP。

六、隐私政策的更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新顶部的生效日期。重大变更时，我们会在 APP 内以适当方式通知你。

本隐私政策的解释和适用适用中华人民共和国法律。
